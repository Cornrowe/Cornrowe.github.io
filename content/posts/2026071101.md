### 第三步：用代码测试连接

这里以Python为例，展示最标准的接入方式：

python
import openai

client = openai.OpenAI(
    api_key="sk-你的云雾API密钥",
    base_url="https://www.yunwuai.cc/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "user", "content": "你好，请用一百字介绍一下你自己。"}
    ],
    stream=True
)

for chunk in response:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")


**特别注意几个坑：**

- `base_url` 必须带 `/v1` 后缀，写成 `www.yunwuai.cc` 不加协议头会失败。
- 必须使用**云雾AI平台生成的API Key**，不要尝试用你从DeepSeek官方拿到的Key。
- `model` 参数务必写对，严格按后台显示走。

### 第四步：验证与排错

运行上述代码，如果没有报错且成功打印出响应，恭喜你——DeepSeek V3.2已经成功接入。

如果出现403或401错误，请按以下顺序检查：
1. API Key是否复制完整（可能是末尾空格没去掉）
2. API Key是否已过期或被删除（在后台重新生成一条再试）
