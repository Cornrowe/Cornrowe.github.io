## 二、30 个高频报错及修复代码

以下报错基于实际使用和官方文档整理，覆盖网络、权限、模型、参数、限流五大类。每个报错附带 **原因** 和 **修复代码**。

### 【网络类】

1. **`ConnectionError: HTTPSConnectionPool`**  
   **原因**：国内服务器无法直接访问海外 API，或 DNS 解析失败。  
   **修复**：确认已使用云雾 API 地址 `https://www.yunwuai.cc/v1`，并检查网络是否能正常访问该域名。  

2. **`requests.exceptions.Timeout`**  
   **原因**：请求超时，可能是模型响应慢或网络延迟。  
   **修复**：增大超时时间（示例：`openai.timeout = 60`），或更换为响应更快的模型（如 GPT-4o-mini）。  

3. **`SSLError: CERTIFICATE_VERIFY_FAILED`**  
   **原因**：SSL 证书验证失败，常见于过时的 Python 环境。  
   **修复**：运行 `pip install --upgrade certifi`，或临时设置 `verify=False`（不推荐生产环境）。  

4. **`ProxyError: HTTPConnectionPool`**  
   **原因**：代码中误配置了代理，导致请求被拦截。  
   **修复**：清除代理设置：`os.environ.pop('http_proxy', None)`。  

5. **`socket.gaierror`**  
   **原因**：域名解析失败。  
   **修复**：检查 DNS 设置，手动 ping `www.yunwuai.cc` 确认可达。  

### 【认证与权限类】

6. **`AuthenticationError: 401`**  
   **原因**：API Key 无效或未传递。  
   **修复**：确保 `api_key` 变量正确，且没有空格。示例：`openai.api_key = "sk-..."`  

7. **`Forbidden: 403`**  
   **原因**：API Key 已被禁用或未激活。  
   **修复**：登录云雾控制台检查密钥状态，重新生成后测试。  

8. **`InsufficientBalance: 402`**  
   **原因**：账户余额不足。  
   **修复**：充值最低 1 元，或利用免费额度测试。  

9. **`InvalidOrganization: 403`**  
   **原因**：组织 ID 不匹配（部分 SDK 要求设置 `openai.organization`）。  
   **修复**：多数情况下不需要设置，若需要，请从控制台获取 Org ID。  

10. **`RateLimitError: 429`**  
    **原因**：请求频率超过限制。  
    **修复**：添加重试机制，或降低并发数（云雾 API 本身不限并发，但特定分组可能有阈值）。  

### 【模型与参数类】

11. **`ModelNotFound: 404`**  
    **原因**：指定模型名称不存在或不支持。  
    **修复**：查看云雾支持的完整模型列表，使用正确的模型 ID（如 `gpt-4o`、`claude-3-opus-20240229`）。  

12. **`InvalidRequestError: 400 - bad request`**  
    **原因**：请求格式错误，如缺少必要字段。  
    **修复**：检查 messages 结构，必须包含 `role` 和 `content`。  

13. **`ContextLengthExceeded: 400`**  
    **原因**：输入 token 数超过模型上下文窗口。  
    **修复**：减少对话历史或使用更大窗口模型（如 `gpt-4-32k`）。  

14. **`InvalidMessageRole: 400`**  
    **原因**：role 只能为 `system`、`user`、`assistant`、`function`。  
    **修复**：修正消息 role 拼写。  

15. **`ImageProcessingError: 400`**  
    **原因**：图片格式不支持或尺寸超限。  
    **修复**：确保图片为 JPEG/PNG，且小于 20MB。使用 base64 编码时不要加多余换行。  

16. **`FunctionCallFormatError: 400`**  
    **原因**：functions 或 tools 参数格式错误。  
    **修复**：参考 OpenAI 官方文档定义函数参数 JSON Schema。  

17. **`ResponseTooLong: 400`**  
    **原因**：`max_tokens` 设置过大，超过模型上限。  
    **修复**：减小 `max_tokens`，或分多次调用。  

18. **`TemperatureOutOfRange: 400`**  
    **原因**：`temperature` 不在 0-2 之间。  
    **修复**：设为 0～2 内的小数，通常 0.7 是合理值。  

19. **`StopSequenceTooLong: 400`**  
    **原因**：`stop` 序列超过 5 个或单个长度超过 256 字符。  
    **修复**：精简停止序列。  

20. **`PresencePenaltyOutOfRange: 400`**  
    **原因**：`presence_penalty` 不在 -2～2 之间。  
    **修复**：调整参数到合法范围。  

### 【流式与并发类】

21. **`StreamError: 500`**  
    **原因**：流式传输中断，可能是网络抖动。  
    **修复**：捕获 `StreamError` 后自动重试，或在非流模式下调用。  

22. **`AsyncClientError: 500`**  
    **原因**：异步请求处理异常。  
    **修复**：确保使用 `asyncio.run` 正确运行异步代码。  

23. **`ConcurrentLimitExceeded: 429`**  
    **原因**：短时间内并发数过高（云雾不限制并发，但个别分组有隐藏限流）。  
    **修复**：加入 `asyncio.Semaphore` 限制并发数量。  

24. **`ChunkedEncodingError`**  
    **原因**：响应分块编码出错，常与流式请求相关。  
    **修复**：关闭流式 `stream=False` 测试，或升级 requests 库。  

### 【工具与集成类】

25. **`ModuleNotFoundError: openai`**  
    **原因**：未安装 openai 库。  
    **修复**：`pip install openai`  

26. **`ValueError: Invalid API Key format`**  
    **原因**：API Key 不是以 `sk-` 开头。  
    **修复**：确认从云雾控制台复制的完整密钥。  

27. **`AttributeError: 'ChatCompletion' object has no attribute 'choices'`**  
    **原因**：响应结构变化，可能是 SDK 版本不兼容。  
    **修复**：降级 OpenAI SDK 到 0.28 以下或升级到 1.x 并调整代码。  

28. **`JSONDecodeError: Expecting value`**  
    **原因**：API 返回非 JSON 内容（如 HTML 502 错误页面）。  
    **修复**：打印原始响应内容 `print(response.text)` 排查。  

29. **`EmptyResponse: the server response was empty`**  
    **原因**：服务器无响应。  
    **修复**：稍后重试，或检查模型是否维护中。  

30. **`MissingSchema: Invalid URL`**  
    **原因**：`base_url` 缺少 `https://` 前缀。  
    **修复**：补全协议 `https://www.yunwuai.cc/v1`。
