### 第二步：获取API Key（并理解密钥的生成逻辑）

登录之后，找到“API密钥”或“API Key管理”这个入口。很多平台会把按钮放在侧边栏或者左上角头像下拉菜单里（云雾也是这样）。

点击“创建新密钥”。注意，这里的操作有一个**关键区别**：

|操作|说明|

|生成密钥+绑定IP白名单|更安全，但如果你用的是动态IP（比如家里的宽带），可能会频繁掉线。|

避坑点：**如果你不确定自己IP是否固定，先不绑定白名单**，等测试通过后再去后台手动添加。很多新手一上来就绑了白名单，结果换到公司网络或者开热点时就404了。

### 第三步：正确配置代码环境（关键是改哪一行）

现在你有Key了，也有免费额度了，该接入代码了。

如果你是直接调用OpenAI的Python SDK，只需要改**两个地方**：

python
import openai

# 关键点1：换API 地址
openai.api_base = "https://www.yunwuai.cc/v1"

# 关键点2：换API Key
openai.api_key = "sk-你的云雾Key"

# 调哪个模型

    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}]
)

print(response.choices[0].message.content)


**深度解析避坑清单**：
- **不要忘记 `v1`**：有人会把 `api_base` 写成 `https://www.yunwuai.cc` 少了一个 `/v1`，然后整天问为什么报错。这个尾缀必须带上。
- **模型名必须写对**：云雾中转站支持500+模型，但模型名要和你后台看到的一模一样。比如你不能写 `gpt-4` 伪装的模型，而是要去后台查看精确的模型ID，通常在菜单“模型列表”里。
- **免费额度只支持部分模型**：如果你用免费额度去调用 `claude-3-opus` 这种高价模型，是跑不通的。至少要先充值1元。
