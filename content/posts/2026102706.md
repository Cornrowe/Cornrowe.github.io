## 第三步：编写核心调用代码

在这个项目里创建一个新文件，比如叫 `index.js`，然后写入下面的代码。这段代码就是你的“第一个对话”核心。

javascript
import OpenAI from "openai";

const openai = new OpenAI({
  // 把这里换成你从云雾平台申请的API Key
  apiKey: "sk-你的云雾API密钥", 
  // 核心！把baseURL指向云雾平台的API地址
  baseURL: "https://www.yunwuai.cc/v1",
});

async function firstConversation() {
  try {
    console.log("正在请求GPT-5.2模型，请稍候...");
    
    const completion = await openai.chat.completions.create({
      // 指定模型为GPT-5.2
      model: "gpt-5.2",
      messages: [
        {
          role: "user",
          content: "用一句中文总结一下，人工智能的未来是什么？",
        },
      ],
      // 设置温度和最大Token数，控制生成内容的随机性和长度
      temperature: 0.7,
      max_tokens: 200,
    });

    // 提取并打印AI的回复
    const responseText = completion.choices[0]?.message?.content || "无内容返回";
    console.log("GPT-5.2的回复：", responseText);
  } catch (error) {
    console.error("请求失败，错误详情：", error.message);
    console.log("建议检查：1.API密钥是否正确 2.余额是否充足 3.网络是否正常");
  }
}

firstConversation();


### 代码逐行解析

- **import OpenAI from "openai"**：导入我们刚才安装的官方库。
- **new OpenAI({...})**：创建一个客户端实例。这里面最关键的就是 **`baseURL`**，你必须把它设置为 https://www.yunwuai.cc/v1。`apiKey`填入你在云雾平台后台拿到的那个Key。
- **async function firstConversation()**：定义一个异步函数，用来发起网络请求。
- **await openai.chat.completions.create({...})**：这是真正的API调用。`model`参数指定了你想用的模型，这里我们填 `gpt-5.2`。`messages`数组里放的就是你的对话内容，结构非常简单：一个用户角色（`role: "user"`）加一句问话（`content`）。
- **completion.choices[0]?.message?.content**：从返回的对象中提取AI给出的回答文本。这是所有OpenAI兼容接口的标准返回结构。
- **错误处理**：我特意加了try...catch块，并带了友好的错误提示。无论你是Key写错了、余额用光了还是网络没连上，它都会告诉你下一步该检查什么。
