#### 第三步：写调用代码（真正的核心操作）

在项目目录创建文件 `index.js`，贴入下面这几行代码：

javascript
const OpenAI = require("openai");

// 配置关键参数
const openai = new OpenAI({
  apiKey: "你的云雾AI-API-Key",  // 替换成你自己的Key
  baseURL: "https://www.yunwuai.cc/v1",  // 云雾API端点地址
});

async function main() {
  try {
    const completion = await openai.chat.completions.create({
      model: "gpt-3.5-turbo",   // API支持全系列模型
      messages: [
        { role: "system", content: "你是一个编程助手，简称‘代码大师’。" },
        { role: "user", content: "帮我用JavaScript写一个排序算法并附上解释。" },
      ],
      max_tokens: 200,
      temperature: 0.7,
    });

    console.log("结果输出：", completion.choices[0].message.content);
  } catch (error) {
    console.error("调用失败：", error);
  }
}

main();


几行代码，不用挂代理，不用手动拼接八爪鱼一样的 header。直接用 `npm start` 或者 `node index.js` 就能运行。3 分钟，对话式 AI 调用就已经跑在你的电脑上了。

1. **Base URL 写错了**：请严格用 `https://www.yunwuai.cc/v1`，多一个 `/` 少一个字母都会报错；
2. **API Key 没有正确替换**：请确认不是你注册完成后忘记复制 Key 了；
3. **模型名写错**：如果你用的是 GPT-4o，那就写 `gpt-4o`，不要写 `gpt-5`；
4. **余额不足**：云雾最低充 1 元人民币，充一下就能接着跑了。

只要 Key 对、URL 对、模型对，就百分百稳定。
