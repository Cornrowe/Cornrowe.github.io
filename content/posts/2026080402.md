### 方案一：OpenAI Java SDK（官方库）

这是最推荐、接入最快的方式。Maven引入openai-java库，将`baseUrl`指向云雾API：

java
import com.openai.client.OpenAIClient;
import com.openai.client.okhttp.OpenAIOkHttpClient;
import com.openai.models.*;

OpenAIClient client = OpenAIOkHttpClient.builder()
    .baseUrl("https://www.yunwuai.cc/v1")
    .apiKey("你的云雾API_KEY")
    .build();

ChatCompletionCreateParams params = ChatCompletionCreateParams.builder()
    .model("ernie-bot-4")
    .addUserMessage("用Java实现一个快速排序")
    .build();

ChatCompletion completion = client.chat().completions().create(params);
System.out.println(completion.choices().get(0).message().content().orElse(""));


**优点**：代码最简洁，流式、函数调用全支持，秒级接入。  
**成本**：完全依赖云雾API定价，1元抵1美元Token，无额外消耗。  
**适用场景**：中小团队、个人开发者，快速验证或生产部署。

### 方案二：Apache HttpClient + 手动构建JSON（原生HTTP调用）

如果你不想引入任何AI专用SDK，直接用HttpClient也能搞定。请求体手动构造：

java
import org.apache.http.client.methods.HttpPost;
import org.apache.http.entity.StringEntity;
import org.apache.http.impl.client.CloseableHttpClient;
import org.apache.http.impl.client.HttpClients;
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;

CloseableHttpClient client = HttpClients.createDefault();
HttpPost post = new HttpPost("https://www.yunwuai.cc/v1/chat/completions");
post.setHeader("Content-Type", "application/json");
post.setHeader("Authorization", "Bearer 你的云雾API_KEY");

String json = "{\"model\": \"ernie-bot-4\", \"messages\": [{\"role\": \"user\", \"content\": \"用Java手写一个单例模式\"}]}";
post.setEntity(new StringEntity(json));

try (var response = client.execute(post)) {
    JsonNode root = new ObjectMapper().readTree(response.getEntity().getContent());
    System.out.println(root.get("choices").get(0).get("message").get("content").asText());
}


**优点**：零依赖，任何Java项目都能用。  
**成本**：同上，但请求响应需要自己处理流式，开发工作量翻倍。  
**适用场景**：老旧项目或对依赖有严格管控的环境。

### 方案三：Spring RestTemplate（Spring生态首选）

Spring家族的项目用RestTemplate或WebClient接入最自然。下面是RestTemplate的示例：

java
import org.springframework.http.*;
import org.springframework.web.client.RestTemplate;
import java.util.*;

RestTemplate rest = new RestTemplate();

HttpHeaders headers = new HttpHeaders();
headers.setContentType(MediaType.APPLICATION_JSON);
headers.setBearerAuth("你的云雾API_KEY");

Map<String, Object> requestBody = new HashMap<>();
requestBody.put("model", "ernie-bot-4");
requestBody.put("messages", List.of(Map.of("role", "user", "content", "用Java实现一个二叉树遍历")));

HttpEntity<Map<String, Object>> entity = new HttpEntity<>(requestBody, headers);
ResponseEntity<Map> response = rest.postForEntity(
    "https://www.yunwuai.cc/v1/chat/completions", entity, Map.class);
System.out.println(response.getBody());


**优点**：与Spring Boot无缝整合，配置化友好。  
**成本**：无附加成本，但需要处理响应解析。  
**适用场景**：Spring微服务架构、企业级Java应用。

---

### 方案四：OkHttp + Gson（轻量高性能）

OkHttp是目前安卓和轻量服务端主流的HTTP客户端，配合Gson解析，性能优秀：

java
import okhttp3.*;
import com.google.gson.*;

OkHttpClient client = new OkHttpClient();
MediaType JSON = MediaType.get("application/json; charset=utf-8");

String json = "{\"model\": \"ernie-bot-4\", \"messages\": [{\"role\": \"user\", \"content\": \"用Java写一个线程安全的计数器\"}]}";
Request request = new Request.Builder()
    .url("https://www.yunwuai.cc/v1/chat/completions")
    .header("Authorization", "Bearer 你的云雾API_KEY")
    .post(RequestBody.create(json, JSON))
    .build();

try (Response response = client.newCall(request).execute()) {
    JsonObject obj = JsonParser.parseString(response.body().string()).getAsJsonObject();
    System.out.println(obj.getAsJsonArray("choices").get(0).getAsJsonObject()
        .getAsJsonObject("message").get("content").getAsString());
}


**优点**：轻量、并行请求性能好。  
**成本**：无额外费用。  
**适用场景**：高并发场景、安卓端集成、IoT设备。
