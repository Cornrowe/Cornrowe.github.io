### 场景二：流式输出（打字机效果）

对于聊天机器人或实时响应的场景，流式输出几乎是必须的。代码变动极小：

python
from openai import OpenAI

client = OpenAI(
    api_key="你的云[云雾ai聚合站](https://www.yunwuai.cc/)API密钥",
    base_url="https://www.yunwuai.cc/v1"
)

# 关键参数：stream=True
stream = client.chat.completions.create(
    model="Qwen-Turbo",
    messages=[
        {"role": "user", "content": "给我写一个快速排序的Python实现，并附上注释。"}
    ],
    stream=True  # 开启流式输出
)

# 逐块接收内容
for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")  # end="" 让输出不换行

## 进阶技巧：错误处理与超时配置

在正式环境中，网络波动和 API 限流是常态。一个好的接入方式必须包含容错机制。

python
from openai import OpenAI, APIError, Timeout, APIConnectionError

client = OpenAI(
    api_key="你的云[云雾ai聚合站](https://www.yunwuai.cc/)API密钥",
    base_url="https://www.yunwuai.cc/v1",
    timeout=30.0  # 设置全局超时时间（秒）
)

try:
    response = client.chat.completions.create(
        model="Qwen-Turbo",
        messages=[{"role": "user", "content": "你好"}],
        max_tokens=200,
        timeout=15.0  # 单次请求超时
    )
    print(response.choices[0].message.content)

except Timeout as e:
    print("请求超时，请检查网络或稍后重试。")
except APIConnectionError as e:
    print("网络连接失败，请确保你的网络环境稳定。")
except APIError as e:
    print(f"API返回了错误：{e}")
except Exception as e:
    print(f"未知错误：{e}")


这段代码覆盖了90%以上的异常情况。你只需要根据自己的业务逻辑，在 `except` 块里加入重试机制或降级策略即可。

## 常见问题与解决

**问题1：报错 “Incorrect API key provided”**

-   **原因**：API 密钥填错了，或者密钥里包含了多余的空格。
-   **解决**：重新复制密钥，确保 `api_key=` 后面没有引号之外的任何空格。如果密钥已丢失，去 云[云雾ai聚合站](https://www.yunwuai.cc/) 控制台重新生成一个。

**问题2：报错 “Model not found” 或 “404”**

-   **原因**：模型名称写错了，或者该模型在平台尚未上架。
-   **解决**：在 云[云雾ai聚合站](https://www.yunwuai.cc/) 的“模型列表”页面确认 `Qwen-Turbo` 是否可用。模型名称严格区分大小写，请确保是 `Qwen-Turbo` 而不是 `qwen-turbo`。

**问题3：请求一直卡住，然后超时**

-   **原因**：网络代理冲突或本地 DNS 问题。
-   **解决**：检查你是否开启了全局代理软件。如果你的系统配置了代理，请为 `api.yunwuai.cc` 或直接在代码中将其加入“直连”列表。最简单的方法是先暂时关闭代理，确认能否联通。

**问题4：响应速度慢，或者偶尔断流**

-   **原因**：模型负载波动或本地网络不稳定。
-   **解决**：启用流式输出（`stream=True`），用户的感知延迟会更低。同时检查你的代码是否对每个请求都新建了 `OpenAI` 客户端，建议复用同一个 `client` 实例以复用连接池。
